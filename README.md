## git활용법
1. working directory, staging area, repository에 대한 개념 이해
   - working directory = 현재 작업중인 영역
   - staging area = 로 update할 녀석들을 선별하는 장소
   - repository = staging area의 녀석들을 으로 최종 저장. 수정 사항을 알기쉽게 텍스트 작성 가능
